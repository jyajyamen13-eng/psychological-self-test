# メンタル＆ソーシャルスキル自己診断ツール

職場復帰・社会生活への準備状況を自分で確認するための自己診断ツールです。

## ツールの目的

このツールは、メンタルヘルスや対人関係スキルに関する自己理解を深めるためのものです。  
就労・社会復帰を目指す方が、現在の状態を整理し、支援者との話し合いに活用することを想定しています。

## 使い方

1. ブラウザで [https://jyajyamen13-eng.github.io/psychological-self-test/](https://jyajyamen13-eng.github.io/psychological-self-test/) にアクセス
2. 氏名を入力し、60問の質問に回答
3. 結果を確認（**利用者向けビュー** / **支援者向けビュー** の2種類）
4. デモデータで実際の出力イメージを事前に確認することもできます（「デモを見る」ボタン）

## 注意事項

- このツールは**医療診断ではありません**
- 結果は参考情報として活用し、**支援者・担当職員と一緒に確認することを推奨**します
- 診断・治療が必要と感じる場合は、専門機関にご相談ください

## プライバシーについて

- 入力データはすべて**ブラウザ内（LocalStorage）に保存**されます
- 外部サーバーへのデータ送信は一切行いません
- ブラウザのデータを消去すれば記録もすべて削除されます

## 動作環境

- モダンブラウザ（Chrome / Firefox / Edge / Safari）
- インターネット接続不要（オフラインでも動作可能）
- サーバー不要・インストール不要
